# HSI 動量突破回測系統 (Streamlit)

1:1 翻譯自 Next.js TypeScript 系統 (224 行 backtest.ts + 2233 行 page.tsx)

## 4 個 Tab

1. **📈 回測** - 跑單次回測, 顯示 6 個子圖 (權益/回撤/價格+通道/買賣點/月度/明細)
2. **🔍 優化** - 網格搜索 5,000 組合, 找最佳參數
3. **📡 即時監察** - 單次信號檢查 + Telegram 推送
4. **📐 公式** - 核心公式文檔

## 5 年回測結果 (預設配置)

- 交易: 25 (勝 16 / 敗 9)
- 勝率: 64.0%
- 年化: -1.05%
- MDD: 13.32%
- 跟 TypeScript 原版一致
