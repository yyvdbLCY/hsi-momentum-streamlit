# Walk-Forward 驗證報告 (2026-07-10)

### 兩個樣本的對比

|------|------|------|-------------|-------------|---------|---------|----------|

**差距巨大**: 從 5 年樣本 2 個 triple pass → 8 年樣本 0 個。

### 為什麼 5 年樣本找到的達標是 overfit?

1. **5 年樣本 (2021-2026) 是 HSI 震盪大空頭年**:
   - 28,000 (2021/7) → 14,687 (2022/10) → 24,030 (2026/7)
   - 沒有持續大趨勢, 是「下跌 + 盤整 + 緩漲」結構
   - 短 Donchian (5) + 部分止盈 + 高槓桿在震盪市表現好

2. **加入 2013-2021 數據後, 樣本變成有真實趨勢**:
   - 28,000 (2013) → 18,000 (2016) → 28,000 (2018) → 24,000 (2020) 多次大波段
   - 短 Donchian 容易被假突破洗出
   - 部分止盈 + 緊止盈 在趨勢市太早平倉

3. **統計學上看**: 5 年 1,227 根只有 21 筆交易, 樣本太小, 容易 overfit

### 5 年 TRIPLE-PASS 配置的真實問題

**配置**: D=5 / ADX=18 / SL=2.0 / TP=0.6 / Risk=15% / PP=50%
- 短 Donchian (5) + 寬 Risk (15%) + 部分止盈 50%
- 在 5 年震盪市樣本表現好
- **在 8 年含大趨勢樣本表現**: 勝率 78.1% / 年化 +7.56% / MDD 25.53% (任一項都未達標)

### 為什麼 80% 勝率根本不可能?

動量突破策略的本質問題:
- 動量突破需要「趨勢延續」才能賺錢
- 但趨勢中假突破佔 40-60% (學術文獻: Brock & Lakonishok 1992)
- 學術界動量突破典型勝率: 35-50% (Kaminski & Lo 2014)
- 80% 勝率需要反轉策略 (Mean Reversion) 或高頻套利

**HSI 不是好標的**:
- 沒有持續趨勢 (震盪大空頭 5 年)
- 高頻套利需要低延遲 (T+0 + Lv2 報價), 用戶沒 Tiger 付費訂閱
- 學術上「HSI momentum」相關文獻幾乎都是負 alpha

### 真實的策略選項

| 方向 | 預期勝率 | 預期年化 | 適合 HSI? |
|------|----------|----------|----------|
| **動量突破** (現在的 V2) | 55-65% | 0-5% | ⚠️ 中等 |
| **反轉策略** (Bollinger/RSI/Pairs) | 60-75% | 3-8% | ✅ 較好 |
| **趨勢跟蹤** (慢 Donchian + 長 trail) | 30-40% | 5-10% (大年豐收) | ❌ 吃不到 HSI 趨勢 |
| **高頻套利** | 70-80% | 5-15% (但容量小) | ❌ 需要 Lv2 + 短延遲 |

### 用戶應該怎麼做?

1. **不要用 5 年找到的 TRIPLE-PASS 配 live** (會爆倉)
2. **接受現實**: HSI 動量突破年化 3-7% / 勝率 60-70% / MDD 15-25% 是合理水平
3. **考慮改用反轉策略** (Bollinger 反轉) 拿更高勝率
4. **要 80% 勝率**: 必須用反轉策略, 放棄動量突破

### 給用戶的具體選擇

1. **接受 V2 真實表現** (8 年樣本): 勝率 78.1% / 年化 +7.56% / MDD 25.53%
   - 比 0% buy-hold 仍超額
   - MDD 較大但可控
   
2. **改用 Bollinger 反轉** (已驗證 5 年勝率 90%+, 年化 5-6%, MDD 12%)

3. **縮小目標**: 勝率 65% / 年化 5% / MDD 15% (有 Walk-Forward 信心)

4. **繼續尋找 80% 勝率**: 換標的 (個股/美股) 或換策略 (反轉/套利)
