## 💡 關鍵發現

1. **D=5 (短週期 Donchian)** + **PP=50% (部分止盈)** 是達標關鍵
2. **TP=0.6 ATR (緊止盈)** 配合 **PP=50%** 形成「鎖利 50% + 留 50% 吃趨勢」的不對稱出場
3. **ADX=18** (中等趨勢閾值) 比預設 22 寬鬆, 容納更多進場機會
4. **Risk=15%** (高槓桿) 放大年化, 但搭配緊止損 SL=2.0 + 部分止盈鎖利, 風險可控
5. **MDD 12.35%** 達標, 證明部分止盈確實控制回撤
6. **21 筆交易** 統計意義足夠, 但仍偏少, 需用 12 年數據 Walk-Forward 驗證
